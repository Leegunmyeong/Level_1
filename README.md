# HTML 생성 후 구조태그 작성 순서
1. `html:5` 쓰고 Tab 자동완성
2. `lang="en"` 을 `"ko"`로 변경하기
3. `title` 제목 작성하기
4. `meta` 작성하고 자동목록 중 `kw`선택하고 키워드 작성하기
    <meta name="keywords" content="교보, 문고, 교보문고"><!-- 검색 키워드(개수제한없음) -->
5. `meta` 작성하고 자동목록 중 `desc` 선택하고 요약 작성하기
    <meta name="description" content="교보문고 회원이 아니신가요? 가입하면 다양한 책 할인혜택이 있습니다."><!-- 사이트 검색 시 나타나는 요약설명 -->
<!-- (위) 각 디바이스(모바일, 태블릿, pc)의 너비에 자동으로 맞추기 -->
6. `body` 내부에 웹의 주요 내용 작성할 준비하기

<!-- 블록 -->
* `<h1>h1~h6</h1>` -> 제목(대중소)
* `<p></p>` -> 내용 작성 or 설명
* `<hr>` -> 구분선 생성
* `<blockquote></blockquote>` -> 긴 인용문

<!-- 인라인 -->
* `<em></em>`-> 중요 부분 글꼴 형태 변환
* `<br>`-> 내용 중바꿈
* `<strong></strong>` -> 중요키워드 강조
* `<q></q>` -> 짧은 인용
* `<del></del>` -> 교체 or 삭제된 내용 표시
* `<s></s>` -> 유효하지않는 내용 
* `<code></code>` 프로그래밍 코드 구분용
* `<sub></sub>`, `<sup></sup>` -> 윗첨자&아래첨자

<!-- 특수문자태그 -->
* `&`: 특수문자태그의 시작기호
* `;`: 특수문자태그의 종료기호
* `&lt;, &gt;`: 왼쪽꺽쇠(<), 오른쪽꺽쇠(>)
